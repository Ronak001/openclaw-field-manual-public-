# OpenClaw Field Manual (Public)

This repo is the **public, approved snapshot** of the *OpenClaw Field Manual* — a practical guide to using OpenClaw in real workflows.

If you’ve ever wondered “How do people actually run an agent across DMs + multiple groups without chaos?” this is the answer.

## What you’ll find here
- Plain‑English mental models (agents, sessions, tools, approvals)
- Practical operating patterns for group + DM usage
- Reliability / debugging playbooks (what breaks, why, and how to fix it)
- Copy/paste workflows and recipes

## What you *won’t* find here (by design)
- No private drafts
- No chat transcripts or direct quotes
- No identifiers (numbers, emails, group IDs, etc.)
- No secrets, tokens, or API keys

Everything is written with strict privacy rules and placeholder examples.
